## ⚙️ How to use the software

Once you install the app, look for a shortcut on your desktop or in your Start menu. Click the icon to open the main window.

The first time you open the program, it asks you to select a folder. Use the folder browser to pick the project you want to clean. The app then scans the files inside that folder. This process takes a few seconds for small projects or a few minutes for larger ones.

After the scan finishes, you see a list of findings. Each item includes a description of the potential improvement. Review the list and select the files you want to fix. If you want to keep a file exactly as it is, you can mark it as a frozen file to prevent changes.

If you like the results, click the "Apply Cleanup" button. The program makes the requested changes to your project. Please note that the program creates a backup folder in your project directory just in case you need to revert any changes.
